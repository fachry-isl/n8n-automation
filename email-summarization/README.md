## Project Description

Inbox overload is a productivity killer. This project solves it with a fully automated pipeline that runs every morning at 7:00 AM, fetching all emails from the past 24 hours and intelligently triaging them using the Eisenhower Matrix framework powered by Google Gemini.

Each email is analyzed and classified into one of four quadrants:

- 🚨 Do First — Urgent & Important (e.g., security alerts, critical updates)
- 📅 Schedule — Not Urgent but Important (e.g., product updates, learning content)
- 🤝 Delegate — Urgent but Not Important
- 🗑️ Delete — Neither Urgent nor Important

The results are aggregated and delivered as a clean, styled HTML email report showing summary stats (total emails, count per quadrant) and grouped email cards per category — making it easy to prioritize your day at a glance.
